zcchen/toolbox
=============================================

A `toolbox` docker image based on Alpine Linux.
This image includes the following tools:
+ `bash`
+ `git`
+ `wget`, `curl`

Example as below:
```
docker run --rm -it zcchen/toolbox bash
```
